# gbainfo

Tools for view and edit gba-roms header

**Usage**
```
./gbainfo romfiles.gba
```

**BIOS file**
Put BIOS file(md5: a860e8c0b6d573d191e4ec7db1b1e4f6) in ~/.config/gbatools/gba_bios.bin or folder with execute file.
